# 프로젝트 설명

법인 카드 사용 승인 절차를 한 곳에서 간단하게! <br>
<img src="./readmeimage/image-14.png">
<br>
상급자는 일반 사원과 관리자의 중간다리 역할을 하게 된다. <br>
일반 사원은 부서 내의 상급자에게 승인을 요청하게 되고, 본인이 부서내 상급자라면 상위 부서의 상급자에게 승인 요청을 하게 된다.
