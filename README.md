### 승인 요청 과정

승인 요청 시 용도 선택의 어려움

<p align="center">

</p>

ai가 사용처와 금액을 통해 자동으로 용도 매핑!

<p align="center">
  <img src="./readmeimage/우리의자랑ai.gif" width="41%">
</p>

알맞게 추천된 결과를 일반 사용자는 상급자에게, 상급자는 재무부에게 승인 요청을 하게 된다.

<p>
